Términos y Condiciones

Al registrarse y usar esta plataforma, aceptas completamente estos términos.

1. Uso Apropiado

Acepta utilizar esta plataforma únicamente para registrar asistencias legales. No se permite acceder a cuentas ajenas, falsificar registros, o realizar actividades fraudulentas.

2. Responsabilidad de Cuenta

Eres responsable de mantener tu contraseña confidencial, proporcionar información precisa y notificar cualquier acceso no autorizado inmediatamente.

3. Privacidad de Datos

Protegemos tus datos personales y no los compartimos con terceros sin consentimiento. Los datos se usan únicamente para fines de control de asistencia conforme a leyes aplicables.

4. Roles de Usuario

Administrador: Acceso completo para crear, editar y eliminar registros.
Usuario: Acceso limitado para registrar asistencia y ver estadísticas.

5. Limitación de Responsabilidad

La plataforma se proporciona "tal cual". No somos responsables por daños derivados del uso o mal uso de la plataforma, información incorrecta del usuario, o pérdida de datos por eventos inesperados.

Al aceptar estos términos, confirmas que has leído y entiendes esta política.
